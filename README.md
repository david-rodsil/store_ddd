# Store API

1. Clonar 
2. Instalar dependencias y paquetes
```
yarn install
```
3. Solicitar, clonar y personalizar el archivo ```.env```
4. Levantar la base de datos en postgres 
```
docker-compose up -d
```
5. Levantar con: 
```
yarn start:dev
```
